# HA Automation Maker

방별·센서별 엔티티 탐색으로 자동화를 쉽게 만드는 한국어 UI 애드온입니다.

거실·안방·작은방 같은 방 단위로, 조명·감지기·환경 센서·난방 같은 카테고리
단위로 엔티티를 찾아가며 "언제(트리거) → 어떤 조건에서 → 무엇을 할지(액션)"를
카드 형태로 조합합니다. 한글 초성 검색(`ㄱㅅ` → 거실)을 지원하고, 저장 전에는
자연어 요약과 실제 생성될 YAML을 함께 보여줍니다.

## 주요 기능

- 트리거 카테고리 위저드로 시작하는 새 자동화 만들기(시간/사람 이동/센서 감지/수치 변화)
- 트리거 · 조건 · 액션을 세로 카드로 구성하는 편집기
- 방 → 카테고리 → 엔티티 3단 피커, 전역 검색(부분 일치 + 초성)
- 자동화 목록에서 켜기/끄기 토글, 수동 실행, 편집, 삭제
- 저장 전 자연어 요약 확인 + YAML 미리보기
- 가스밸브·잠금장치 등 안전 관련 동작 포함 시 확인 경고 표시

자세한 사용법은 **문서(Documentation)** 탭을 참고하세요.

이 애드온은 Home Assistant Core API(`homeassistant_api`)만 사용하며, 별도의
설정 옵션이 없습니다. Ingress를 통해 사이드바 패널로 노출됩니다.
